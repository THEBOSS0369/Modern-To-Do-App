# Modern To Do App

### Organize your daily task with this Modern To Do App.

# Key Features:

### Next.js 15+ with TypeScript and modern practices
### Beautiful dark theme with glassmorphism effects and gradient accents
### Smooth animations with Framer Motion (fade-in/out, scale, slide effects)
### Full CRUD functionality - Create, Read, Update, Delete tasks
### Advanced task management - Status tracking, priority levels, due dates
### Search & Filter - Find tasks quickly
### Statistics dashboard - Visual progress tracking
### Responsive design - Works perfectly on all devices
### Local storage - Tasks persist between sessions

# File Structure:
### All code is organized in the src/ folder with proper separation:
### src/app/ - Next.js app router files
### src/components/ - Reusable UI components
### src/types/ - Type definitions
